#  E-Pharmacy — Online Medicine Ordering System

A full-stack web application for ordering medicines online, built as a 
final year graduation project. It allows users to browse medicines, 
upload prescriptions, manage their cart, and place orders — while giving 
admins full control over products, prescriptions, and order management.

---

## 🚀 Features

### User Side
- User registration and login with hashed passwords
- Browse medicines by category (Ayurveda, Baby Care, Oral Care, etc.)
- Search for medicines
- Add to cart and wishlist
- Upload prescription for prescription-based medicines
- Checkout with delivery details
- View and cancel orders
- Submit product reviews and feedback
- Forgot password via email reset link

### Admin Side
- Secure admin login
- Dashboard with order and user stats
- Add, update, and delete products
- Manage and approve/reject prescriptions
- View and update order statuses
- Manage payments
- View and delete user feedback
